# Unity.OS

OS / Platforms / Devices utility functions for Unity

## Installation

Add the dependency to your `manifest.json`

```json
{
  "dependencies": {
    "jd.boiv.in.os": "https://github.com/starburst997/Unity.OS.git"
  }
}
```

## Usage

```csharp
var isTablet = OSUtils.IsTablet();
```
